# LORA-CURRICULUM

Curriculum Learning with Meta-Token Differences for LoRA Fine-tuning

## Dataset
- 98,057 reasoning task samples from 6 datasets
- BoolQ, ARC (Easy/Challenge), OpenBookQA, Hellaswag, Winogrande

## Components
- Meta-token difference based difficulty measurement
- Curriculum scheduler with 3 stages
- Training pipeline with LoRA fine-tuning
